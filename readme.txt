Project Name: Virtual Soccer (for the lazy partner)
This game allows your energetic friend to control a virtual team to play a full game of soccer against another player, who is permitted to relax and control their team with a keyboard!

Instructions for Play:
The user must connect a Kinect 2 to the Windows computer being used. Then, the user must open all three python files in a single window of the Microsoft Visual Studio IDE. Then run the file 'game_main.py'. 
To select an option (controls or play game), the user must make a fist over the desired option onscreen.

Libraries needed: pykinect2, pygame
Hardware needed: Kinect 2
